# ReadHandler

A safe file reader api.

# Installation

Install the library

`em list`
`em install`
`em build lib`

## Usage

```c
#include "ReadHandler.h"

int main(void) {
  char *line1;
  char *line2;
  struct read_handler *h = read_handler_new();

  if(read_handler_open(h, "file.txt")) {
    line1 = read_handler_read_line();
    line2 = read_handler_read_line();
  }

  printf("%s\n", line1);
  printf("%s\n", line2);

  free(line1);
  free(line2);

  read_handler_close(h);
}
```
